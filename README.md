# Shopping Cart with Localization
A java console application that calculates shopping cart total cost with multi-language support.

## Features
- Calculate total cost
- Localization support: English, Finnish, Swedish, Japanese
- Unit tests with JaCoCo coverage
- CI/CD pipeline with Jenkins
- Dockerized application

## How to Run

### Run with Docker
```bash
docker pull nurha2024/otp2_week1_assignment:latest
docker run -it nurha2024/otp2_week1_assignment:latest
```
